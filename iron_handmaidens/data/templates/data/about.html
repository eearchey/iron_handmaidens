Iron Handmaidens

Emily Archey, Josh Hourigan, Sean Macander, Alden Moreton, Vyas Padmanabhan

This application takes in a .csv or .mat file full of EMG data and processes the data to give the user a graphical visualization.


Usage

Home Page
 Upload files
   Upload one or multiple .csv OR .mat files of EMG data. This will take you to the visualize page.
 About page
  Go to the about page
 Home page
  Go back to the beginning of the process

Visualize Page
 The data you uploaded will be presented in graphical form, along with certain important data points.
 The graphs can be manipulated as desired.
 You can also upload new data on this page.
 From the visualize page, you can also go to the about page or back to the home page.


Running Locally

To run this program locally, navigate to the iron_handmaidens folder with manage.py inside. From there, run the following command:
  python3 manage.py runserver
 In the terminal, the local URL should be displayed. Paste this into your browser of choice (Chrome, Safari, etc.)
 You are now in the application!
